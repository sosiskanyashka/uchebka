#Game website

##Описание и предметная область
Этот проект представляет собой сайт с видеоиграми с регистраицей, авторизацией и списком популярных игр, написанный на php с использованием pdo, в качесттве базы данных был выбран MySQL. Сайт удобен и подходит для отслеживания новых и популярных игр клиентом.
