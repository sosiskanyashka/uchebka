# Отчет по созданию сайта-визитки

## 1. Цель задания

Целью данного задания было приобретение практических навыков в:

*   Создании веб-страниц.
*   Освоении семантической разметки HTML.
*   Применении стилей с использованием CSS.
*   Внедрении простейших интерактивных элементов с помощью JavaScript (опционально).

## 2. Технические требования и реализация

**2.1. Структура страницы:**

Сайт-визитка реализован в соответствии со следующей структурой:

*   **Шапка (Header):**
    *   Заголовок с именем владельца визитки: "Иван Иванов".
    *   Краткое описание деятельности: "Frontend-разработчик | Веб-дизайнер".
*   **Раздел «О себе» (Section #about):**
    *   Текстовое описание, рассказывающее о владельце сайта.
    *   Фотография (опционально): `profile.jpg`.
*   **Раздел «Портфолио/Навыки» (Section #portfolio):**
    *   Перечень навыков: HTML5, CSS3, JavaScript, адаптивная верстка, Git.  Представлен в виде списка `<ul>`.
*   **Раздел «Контакты» (Section #contacts):**
    *   Информация для связи:
        *   Электронная почта: `ivan.ivanov@example.com` (ссылка `mailto`).
        *   Ссылки на социальные сети: VK, LinkedIn (открываются в новых вкладках).
*   **Подвал (Footer):**
    *   Дополнительная информация (копирайт): © 2024 Иван Иванов. Все права защищены.

**2.2. Технические требования и реализация:**

*   **HTML-разметка (index.html):**
    *   Использованы семантические теги: `<header>`, `<main>`, `<section>`, `<footer>`.
    *   Включены мета-теги для кодировки (UTF-8) и адаптивности (viewport).
    *   Подключен файл `styles.css` для стилизации.
    *   Подключен файл `script.js` для интерактивности (опционально).

*   **Стилизация с помощью CSS (styles.css):**
    *   Определены базовые цвета, шрифты, отступы и выравнивание для различных элементов (body, header, main, section, h2, ul, li, a, img, footer).
    *   Реализована адаптивность дизайна с использованием медиа-запросов `@media (max-width: 768px)` для экранов меньшего размера.
    *   Использован Flexbox для компоновки элементов на странице (в основном для центрирования и организации элементов в разделах).

*   **Интерактивность с помощью JavaScript (script.js) (по желанию):**
    *   Реализовано изменение цвета заголовка при наведении курсора мыши (mouseover/mouseout).
    *   Реализовано всплывающее окно (alert) при нажатии на ссылку с `mailto:` (для демонстрации базовой функциональности).
    *   Использованы обработчики событий (addEventListener) для взаимодействия с элементами DOM.

*   **Структура проекта:**
    *   Все файлы (HTML, CSS, JS, изображение) организованы в единой папке проекта.

## 3. Этапы выполнения

1.  **Планирование и проектирование:**
    *   Разработана схема структуры сайта (см. раздел 2.1).

2.  **Разработка:**
    *   Создана базовая HTML-разметка в файле `index.html`.
    *   Добавлена стилизация с использованием CSS в файле `styles.css` для обеспечения современного внешнего вида и адаптивного дизайна.
