# Отчет по лабораторной работе №1: Электронные книги (C++)

## 1. Постановка цели

Целью данной лабораторной работы является закрепление навыков работы со структурами в языке C++, включая создание, ввод данных, вывод данных, динамическое выделение памяти и работу с векторами.  В ходе работы необходимо реализовать программу, моделирующую электронные книги, с возможностью создания, хранения и отображения информации о них.

## 3. Отражение задач

В рамках лабораторной работы были выполнены следующие задачи:

*   **Разработка структуры `Ebook`:** Определена структура `Ebook`, содержащая поля `manufacturer` (фирма-производитель), `screenSize` (размер экрана) и `hasBacklight` (наличие подсветки).
*   **Создание экземпляров структуры:** Создано два экземпляра структуры `Ebook` в статической памяти.
*   **Функция ввода данных (`inputEbook`):** Реализована функция `inputEbook`, принимающая указатель на структуру `Ebook` и заполняющая поля структуры данными, введенными с консоли.
*   **Функция вывода данных (`printEbook`):** Реализована функция `printEbook`, принимающая экземпляр структуры `Ebook` и выводящая информацию о книге на консоль.
*   **Динамическое создание и хранение в векторе:**  Реализована функция `createEbooksAndAddToVector`, создающая динамические экземпляры структуры `Ebook` и сохраняющая указатели на них в векторе `ebookVector`.  Реализована функция `freeEbookMemory` для освобождения выделенной памяти.
*   **Реализация псевдоменю:** Создано псевдоменю, позволяющее пользователю выбирать различные действия с книгами (вывод информации о статических книгах, создание и вывод информации о динамических книгах, освобождение памяти).

## 6. Вывод

В ходе выполнения данной лабораторной работы я научился:

*   **Работать со структурами в C++:**  Создавать структуры, определять поля, создавать экземпляры структур.
*   **Использовать функции для ввода и вывода данных:**  Реализовывать функции, принимающие структуры в качестве аргументов и выполняющие операции ввода/вывода.
*   **Динамически выделять память:** Использовать операторы `new` и `delete` для выделения и освобождения памяти под объекты.
*   **Работать с векторами:**  Использовать векторы для хранения динамически созданных объектов.
*   **Обрабатывать память для динамических объектов:**  Понимать важность освобождения памяти, выделенной динамически, и реализовывать функции для этого.
*   **Создавать псевдоменю:**  Разрабатывать простой интерфейс для взаимодействия с программой.

Новым для меня было:

*  Более глубокое понимание управления памятью в C++ и необходимость ручного освобождения памяти при использовании `new`.
* Практическое применение векторов для хранения указателей на динамические объекты.
* Применение `getline` для корректного ввода строк с пробелами.
